项目简介

FaceSwap是一个基于Python和PyQt5开发的实时人脸交换应用，利用先进的深度学习技术，可以在视频流中实时替换人脸。该应用提供了直观的图形用户界面，使用户能够轻松地进行人脸交换操作，并支持视频录制和截图功能。

功能特点

实时人脸交换

通过摄像头实时捕捉并替换视频流中的人脸，提供流畅的换脸体验

预设人脸选择

内置多个预设人脸（马保国、马斯克、赵丽颖），一键选择即可使用

自定义人脸上传

支持上传自定义人脸图片作为替换源，满足个性化需求

视频录制

可录制人脸交换后的视频并保存，方便分享和回顾

截图功能

支持拍摄当前画面并保存为图片，捕捉精彩瞬间

全屏预览

支持全屏模式查看效果，获得更沉浸的体验

安装说明

环境要求

  • Python 3.7+
  • OpenCV
  • PyQt5
  • InsightFace
  • NumPy
硬件推荐

为了获得最佳体验，我们推荐以下硬件配置：

  • CPU: Intel i5 或更高
  • 内存: 8GB 或更高
  • 显卡: 支持CUDA的NVIDIA显卡（可选，但能显著提升性能）
  • 摄像头: 720p或更高分辨率

安装步骤

1

克隆仓库到本地

git clone https://github.com/elon-zz/BigDataCourse-FaceSwapVideo.git
cd FaceSwapVideo
2

安装依赖

pip install -r requirements.txt
3

下载模型文件

从 Hugging Face 下载 inswapper_128.onnx 模型文件，并将其放置在 models 目录下。

mkdir -p models
# 下载模型文件并放入models目录

使用方法

1

启动应用

cd face_swap_ui
python main.py
2

使用步骤

  • 从右侧选择一个预设人脸或上传自定义人脸
  • 点击"启动摄像头"按钮开始实时人脸交换
  • 可以点击"开始录制"按钮录制视频
  • 可以点击"拍摄截图"按钮保存当前画面
  • 录制的视频保存在output_videos目录
  • 截图保存在output_photos目录

使用技巧
  • 确保光线充足，面部清晰可见，以获得最佳效果
  • 使用全屏模式可以获得更好的视觉体验
  • 录制视频时，可以通过进度条查看录制时间
  • 自定义人脸图片应当包含清晰的正面人脸
  • 如果检测不准确，尝试调整姿势或光线条件

技术实现

本项目基于以下技术实现：

InsightFace
PyQt5
OpenCV
多线程处理

核心技术

  • InsightFace：用于人脸检测和识别，提供高精度的人脸特征提取
  • PyQt5：构建图形用户界面，提供丰富的UI组件和事件处理机制
  • OpenCV：处理视频流和图像，提供高效的图像处理功能
  • 多线程处理：使用QThread实现摄像头捕获和处理的并行执行，提高性能

实现细节

本项目的核心实现包括：

  1. 使用InsightFace的ONNX模型进行人脸检测和特征提取
  2. 通过PyQt5构建用户友好的界面，实现各种交互功能
  3. 利用OpenCV处理摄像头视频流，实现实时图像处理
  4. 采用多线程架构，将视频捕获和处理分离，提高应用响应速度
  5. 实现视频录制和图像保存功能，方便用户分享使用效果

常见问题

解决方案：从Hugging Face下载inswapper_128.onnx模型文件，并放置在models目录下。

确保模型文件名称正确，且放置在正确的目录中。应用程序会在启动时检查该文件是否存在。

解决方案：确保您的摄像头设备正常工作，并且没有被其他应用程序占用。

  • 检查摄像头是否已正确连接到计算机
  • 关闭可能正在使用摄像头的其他应用程序
  • 在系统设置中确认摄像头权限已开启
  • 尝试重启应用程序或计算机

解决方案：尝试在光线充足的环境中使用，并确保面部清晰可见。

  • 调整光线条件，避免过暗或强光直射
  • 保持面部正对摄像头，减少侧脸角度
  • 避免面部被头发、眼镜等物品遮挡
  • 保持适当距离，不要离摄像头太近或太远

解决方案：检查计算机配置是否满足要求，并关闭不必要的后台程序。

  • 关闭其他占用资源的应用程序
  • 降低摄像头分辨率设置
  • 如有NVIDIA显卡，确保已安装最新驱动
  • 考虑升级计算机硬件配置
